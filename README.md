# DevOps Portfolio - Jatin Mittal

A modern, professional portfolio website showcasing DevOps expertise, built with HTML, CSS, and JavaScript.

## 🚀 Features

- **Particle Background Animation**: Interactive canvas-based particle system
- **Glassmorphism Design**: Modern frosted glass effects with dark theme
- **Responsive Layout**: Fully responsive across all devices
- **Smooth Animations**: Scroll-triggered fade-in animations
- **Project Filtering**: Dynamic filtering for project categories
- **Typing Animation**: Auto-typing hero text with rotating messages
- **Professional Projects**: Showcasing real-world DevOps achievements

## 🛠️ Tech Stack

**Frontend:**
- HTML5 (Semantic markup)
- CSS3 (Glassmorphism, animations, flexbox/grid)
- Vanilla JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter, JetBrains Mono)

**Deployment:**
- GitHub Pages ready

## 🚦 Getting Started

### Local Development

1. Clone this repository or download the files
2. Open `index.html` in your browser
3. That's it! No build process needed.

### Customization

**Update Personal Information:**
- Edit `index.html` to update name, bio, contact details
- Replace profile image at `assets/images/profile.jpg`
- Update social media links in hero and footer sections

**Modify Projects:**
- Edit project cards in `index.html` (Projects Section)
- Update project images in `assets/images/`
- Adjust metrics and descriptions to match your work

**Change Colors:**
- Modify CSS variables in `style.css` (`:root` section)
- Update accent colors to match your brand

## 🌐 Deployment

### GitHub Pages

1. Create a new repository named `yourusername.github.io`
2. Push all portfolio files to the repository
3. Go to Settings → Pages
4. Select branch `main` and folder `/ (root)`
5. Your site will be live at ` https://yourusername.github.io`

## ✨ Features Breakdown

### JavaScript Modules
- **ParticleSystem**: Canvas-based particle animation
- **TypingAnimation**: Auto-typing text effect
- **Navigation**: Smooth scroll and active section highlighting
- **ScrollAnimations**: Intersection Observer for fade-in effects
- **ProjectFilter**: Dynamic project filtering by category
- **ImageHandler**: Graceful fallback for missing images

### Performance Optimizations
- Debounced scroll events
- RequestAnimationFrame for smooth animations
- Lazy loading considerations
- Efficient DOM manipulation
